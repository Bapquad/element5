Introduction

element5 is a static class has follow methods:

element5.Create( target )

Creating the HTML element with the query description string.

The param

  • target:string a description string in the element query format.

How to use

  • var el0 = element5.Create( 'div#wrapper' );
  • var el1 = element5.Create( 'div.wrapper' );
  • var el2 = element5.Create( 'div[id=wrapper][class=layout]' );

The return

The DOM element.

element5.GetBody()

Receives the body element. Notes with the DOMElementLoaded event type.

How to use

  • window.addEventListener( 'DOMElementLoaded', function( e )
  • {
  •     var body = element5.GetBody();
  • }, false );

The return

The BODY element.

element5.GetWindow()

Receives the window.

How to use

  • var winel = element5.GetWindow();
  • console.log( winel );

The return

The window element.

element5.MountDevice( description )

Mounts the media device specification which your apps implements with.

The param

  • description:object the description object describies the media device.

How to use

  • var mobile400 =
  • {
  •     deviceOption: '',
  •     deviceType: 'screen',
  •     features: [ 'max-width:399px' ],
  •     producted: false
  • }
  • element5.MountDevice( mobile400 );

The return

The device description object.

element5.Extension( extObj )

Allow extends to add more functions for DOM element. Note, it helps to extends, but it isn't help to override.

The param

  • extObj:object the extension object contains the mothods used to manipulate on the DOM elements.

How to use

  • element5.Extension( {
  •     sayHello: function()
  •     {
  •         console.log( 'Hello! I\'m an element.' )
  •     }
  • } );
  • var dom = element5.Create( 'h1' );
  • dom.sayHello();

The return

The void return.
