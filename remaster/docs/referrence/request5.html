request5

Introduction

request5 is a static class about network such as get, post ajax.

Functions

request5.Initialize( url, data )

Initializing a request object. This request object is instance of XMLHttpRequest.

The params

  • url:string the url string to request.
  • data:object the data package to request.

How to use

  • var myXHR = request5.Initialize( 'http://examples.com' );
  • console.log( myXHR );

The return

A XMLHttpRequest instance.

request5.Instance( url, data )

Creates a new request instance. This request object is instance of XMLHttpRequest.

The params

  • url:string the url string to request.
  • data:object the data package to request.

How to use

  • var myXHR = request5.Instance( 'http://examples.com' );
  • console.log( myXHR );

The return

A XMLHttpRequest instance.

request5.Connect( url, onload, data )

Initializing a new request instance then make a request to url. This request object is instance of XMLHttpRequest.

The params

  • url:string the url string to request.
  • onload:function the event handle occurs when a request has completely loaded.
  • data:object the data package to request.

How to use

  • var myConn = request5.Connect( 'http://examples.com', function()
  • {
  •     // do something ...
  • });
  • console.log( myConn );

The return

A XMLHttpRequest instance.

request5.LoadVariables( url, onload, data )

Initializing a new request instance then make a request to url. This request object is instance of XMLHttpRequest and extended by a object called DataModifier.

The params

  • url:string the url string to request.
  • onload:function the event handle occurs when a request has completely loaded.
  • data:object the data package to request.

How to use

http://examples.com/variables.txt

  • username=Jack Sander&career=Developer&age=21
  • var myVars = request5.Connect( 'http://examples.com/variables.txt', function()
  • {
  •     // do something ...
  • });
  • console.log( myVars );

The return

A XMLHttpRequest instance.

XHR Modifier Methods

Connect( syncFlag )

Make a connect.

The param

  • syncFlag:boolean a flag specifies whether be or not wait the response. Default it is true value.

How to use

  • var myConn = request5.Initialize( 'http://examples.com/variables.txt' );
  • myVars.Connect();
  • console.log( myConn );

The return

The self XMLHttpRequest instance.

Abort()

How to use

  • console.log( myVars );

The param

How to use
