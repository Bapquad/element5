style5

Introduction

style5 is a static class has a lot of static functions and the dynamic methods.

Functions

style5.AddLine( selector )

Creating the style rule element.

The param

  • selector:string a description string in the element query format.

How to use

  • var styleObject = style5.AddLine( 'div#wrapper' );
  • console.log( styleObject );

The return

The self style rule object.

style5.AddCss( selector )

Add the style rule element.

The param

  • selector:string a description string in the element query format.

How to use

  • var styleObject = style5.AddCss( 'div#wrapper' );
  • console.log( styleObject );

Methods

AddSelector( selector )

Add more the selector to the style rule.

The param

  • selector:string a description string in the element query format.

How to use

  • var styleObject = style5.AddCss( 'div#wrapper' );
  • styleObject.AddSelector( '#header' );
  • console.log( styleObject );

The return

The self style rule object.

AddStyle( name, value )

Add more the style properties to the style rule.

The params

  • name:string the name of the style property.
  • value:string the value of the style property.

How to use

  • var styleObject = style5.AddCss( 'div#wrapper' );
  • styleObject.AddStyle( 'color', '#000' );
  • console.log( styleObject );

The return

The self style rule object.
