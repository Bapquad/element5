geo5

Introduction

geo5 is static class about global position system.

Functions

geo5.GetPosition()

Receives the location of device.

How to use

  • geo5.onDetect( function( e )
  • {
  •     console.log( e.position.coords.latitude, e.position.coords.longitude );
  • });
  • var geoObj = geo5.GetPosition();
  • console.log( geoObj );

geo5.ScanWith( success, error, options )

Scans current position of the device.

The param

  • success:function event handle occurs if get location successfully.
  • error:string event handle occurs if get location failure.
  • options:object options for scan.

How to use

  • var geoObj = geo5.ScanWith( function( e )
  • {
  •     console.log( e.position.coords.latitude, e.position.coords.longitude );
  • });
  • console.log( geoObj );

The return

The geo5 object.
