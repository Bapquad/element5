The params

  • DOMElement:DOMObject a html dom object.
  • DOMAddEventHandle:function a event handle occurs when DOM added into document.
  • DOMRemoveEventHandle:function a event handle occurs when DOM removed into document.

How to use

  • var formElement = element5( '#formID' );

  • {
  •     console.log( e );
  • });

The void return.
