media5

Introduction

media5 is a static class has follow static functions.

Functions

media5.StartCamera( video, width, height )

Starts the camera if it has supported by user device.

The param

  • video:DOM Element a DOM element by video tag.
  • width:DOM Element the length of width of the camera.
  • height:DOM Element the length of height of the camera.

How to use

  • var video = element5.Create( 'video#camera' );
  • var camera = media5.StartCamera( video, 320, 240 );
  • console.log( camera );

The return

A camera object instance.

media5.StartMicrophone()

Starts the microphone if it has supported by user device.

How to use

  • var mic = media5.StartMicrophone();
  • console.log( mic );

The return

A microphone object instance.
