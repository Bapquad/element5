motion5

Introduction

motion5 is a static class has a lot of static functions below.

Functions

Sprite( width, height, imageURL )

Creates the sprite element.

The param

  • width:number the number presents the length of the sprite's width.
  • height:number the number presents the length of the sprite's height.
  • imageURL:string the string presents for image resource path.

How to use

  • var sprite = motion5.Sprite( 50, 45, '/images/sprite.png' );
  • var player = element5.Create( 'p#player' );
  • sprite.Effect( player );

The return

A Sprite object instance.

Sprite class

Run( fps )

Run a sprite animation with the fps.

The param

  • fps:number the number presents the framerate of the sprite's animation. Default, it is set 30 value.

How to use

  • var sprite = motion5.Sprite( 50, 45, '/images/sprite.png' );
  • var player = element5.Create( 'p#player' );
  • sprite.Effect( player );
  • player.MotionSprite.Run();

The return

A Sprite object instance.

Stop()
