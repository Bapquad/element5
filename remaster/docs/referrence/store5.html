store5

Introduction

store5 is a static class about data storage such as localstore, sessionstore, and indexDB.

Functions

store5.Open( storageType )

Open a database simulator.

The param

  • storageType:string a string present for database type.

How to use

  • var db = store5.Open( 'cookie_ds' );
  • console.log( db );

The return

The storage object value. The storageType param has follow constants:

  • 'cookie_ds' - data storage base on cookie. It returns the ExpireData Simulator object.
  • 'local_ds' - data storage base on local storage interfaces. It returns the LocalData Simulator object
  • 'session_ds' - data storage base on session storage interfaces. It returns the SessionData Simulator object
  • 'indexed_db' - data storage base on IndexedDB interfaces. It returns the IndexedDB Simulator object, It must be declared with a database name.

ExpireData Simulator Methods

public( name, value, expire )

Creates the public variable. This can access everywhere in the apps.

The params

  • name:string a string presents the name of variable.
  • value:any (Optional) a data value.
  • expire:number (Optional) a number presents the expire time. In the second unit.

How to use

  • var cookieDB = store5.Open( 'cookie_ds' );
  • cookieDB.public( 'username', 'Huy Cuong', 24 * 60 * 60 );

The return

The ExpireData Record Item object.

private( name, value, expire )

Creates the private variable. This can access in the current branch only.

The params

  • name:string a string presents the name of variable.
  • value:any (Optional) a data value.
  • expire:number (Optional) a number presents the expire time. In the second unit.

How to use

./sub_branch/index.html

  • var cookieDB = store5.Open( 'cookie_ds' );
  • cookieDB.private( 'username', 'Huy Cuong', 24 * 60 * 60 );
  • console.log( cookieDB.private( 'username' ) );

./index.html

  • var cookieDB = store5.Open( 'cookie_ds' );
  • console.log( cookieDB.private( 'username' ) );

The return

The ExpireData Record Item object.

IndexedDB Simulator Methods

The param

  • tables:object the table structures.

How to use

  • var indexDB = store5.Open( 'indexed_db', 'program' );
  • indexDB.Init(
  •     {
  •         'name': 'mission_one',
  •         'key': 'id',
  •         'structure': [
  •             {
  •                 'name' : 'player_name',
  •                 'key' : [ 'playerName', 'playerBirth' ],
  •                 'types' : { 'unique' : true }
  •             },
  •         ],
  •     }
  • );

The IDBOpenDBRequest object with IndexedDB Simulator extensions.
