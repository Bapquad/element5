geo5.Scan()

Find position of device.

How to use:

geo5.GetLocation();

geo5.ScanWith( success, error, options )

Find position of device with params.

  • success - Function handle called when found out the position.
  • error - Function handle called when error occurs.
  • options - Options for scan.

geo5.Reset()

Reset the find position session for new search.

geo5.onDetect( handle )

Handle called when position be found.

  • handle - Function handle.

geo5.onChange( handle )

Handle called when position changes.

  • handle - Function handle.
