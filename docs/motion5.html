motion5 Object

motion5.Sprite( width, height, imageURL )

Find or create the timeline with the description in timeline param.

  • width - The sprite width.
  • height - The sprite height.
  • imageURL - The image URL string of sprite.

How to use:

var sprite = motion5.Sprite( 40, 75, '/images/player_sprite.jpg' ); 
var player = element5.Create( 'p#player' );
sprite.Effect( player );

motion5.Sprite.Run( fps )

Run animation in fps.

  • fps - The FPS of animation.

How to use:

var sprite = motion5.Sprite( 40, 75, '/images/player_sprite.jpg' ); 
var player = element5.Create( 'p#player' );
sprite.Effect( player ); 
player.MotionSprite.Run();

motion5.Sprite.Action( topIndex )

  • topIndex - The index of action.
