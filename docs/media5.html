media5 Object

media5.StartCamera( video, width, height )

Start the camera of device.

How to use:

var video = element5.Create( 'video#camera' ); 
var cam = media5.StartCamera( video, 240, 320 );

media5.StartMicrophone()

Start the microphone of device.

How to use:

var mic = media5.StartMicrophone();
