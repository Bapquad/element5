request5 Object

request5.Initialize( url, data )

Initializing the request.

  • url - The url for request.
  • data - The data for request.

How to use:

var request = request5.Initialize( 'http://examples.com' );

request5.Connect( url, onload, data )

connect the request.

  • url - The url for request.
  • onload:function - The function called when load complete.
  • data - The data for request.

How to use:

var request = request5.Connect( 'http://examples.com' );

request5.LoadVariables( url, onload, data )

Load the variable from the request.

  • url - The url for request.
  • onload:function - The function called when load complete.
  • data - The data for request.

How to use:

var request = request5.LoadVariables( 'http://examples.com/variables.txt' );

request5.onAbort( handle )
