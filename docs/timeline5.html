timeline5 Object

timeline5.Create( timeline )

Find or create the timeline with the description in timeline param.

  • timeline - Timeline string of searching.

How to use:

var timeline = timeline5.Create( 
{
    name: 'leftToR', 
    keyframes: 
    [
        { time: '0', style: { transform: 'translateX( -100% )', }, }, 
        { time: '100', style: { transform: 'translateX( 0px )', }, }, 
    ], 
}); 
var el = element5( '#aside' );
el.Play( timeline ); // Or timeline.Effect( el, duration, delay, timeing )
