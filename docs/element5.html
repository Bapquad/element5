element5 Object

element5( target, limit )

Find or create the elements with the description in target.

  • target - Selector string of searching.
  • limit - Creation limit. Default it is set to 10 value.

How to use:

var el = element5( '#wrapper' );

Climb Modifier

Find ( selector )

Find the elements which be inside. This method has one param.

  • selector - selector string of searching.

How to use:

var el = element5( '#wrapper' );
var result = el.Find( '.search-item' ); 
console.log( result );
