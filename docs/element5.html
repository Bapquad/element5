element5 Object

element5( target, limit )

Find or create the elements with the description in target.

  • target - Selector string of searching.
  • limit - Creation limit. Default it is set to 10 value.

How to use:

var el = element5( '#wrapper' );

Climb Modifier

Find( selector )

Find the elements which be inside. This method has one param.

  • selector - selector string of searching.

How to use:

var el = element5( '#wrapper' );
var result = el.Find( '.search-item' ); 
console.log( result );

Next()

Find the element which be next of.

How to use:

var el = element5( '#wrapper' );
var result = el.Next(); 
console.log( result );

Prev()

Find the elements which be previous of.

How to use:

var el = element5( '#wrapper' );
var result = el.Prev(); 
console.log( result );

Parent()

Find the element which contains it.

How to use:

var el = element5( '#wrapper' );
var result = el.Parent(); 
console.log( result );

Parents( selector )

Find the parent like as selector. This method as one param.

  • selector - Selector for parent node which wants to select.

How to use:

var el = element5( '#wraper.layout' );
var result = el.Parents( '#wrapper' ); 
console.log( result );

Style Modifier

css( styles )

css( propertyName, propertyValue )

Set the css style for element. This method has one param.

How to use alt-method:

var el = element5( '#wraper.layout' );
el.css( 'color', '#000' ); 
el.css( 'background', '#fff' );

Equip( element )

Equips the element. This method has one param.

  • element - The element which wants to equip into.

How to use:

var top = element5.Create( 'h2[id=top]' );
var head = element5( '#header' );
head.Equip( top );

EquipedBy( element )

Equips the element by self. This method has one param.

  • element - The element which wants be equiped by.

How to use:

var top = element5.Create( 'h2[id=top]' );
var head = element5( '#header' );
top.EquipedBy( head );

EquipEvent( addHandle, removeHandle )

Fire when element added or remove another DOM element. This method has two params.

  • addHandle - The handle which fires when the another node element added to this element.
  • removeHandle - The handle which fires when the another node element removed from this element.

How to use:

var top = element5.Create( 'h2[id=top]' );
var head = element5( '#header' );
head.EquipEvent( function( e ) { console.log( 'Node Added!' ) }, function( e ) { console.log( 'Node Removed' ) } );
top.EquipedBy( head );

EquipBefore( element )

Equips this element before the element which specify in param. This method has one param.

  • element - The element which wants to equips before with.

How to use:

var top = element5.Create( 'h2[id=top]' );
var option = element5( '#header #option' );
top.EquipBefore( option );

EquipAfter( element )

Equips this element after the element which specify in param. This method has one param.

  • element - The element which wants to equips after with.

How to use:

var top = element5.Create( 'h2[id=top]' );
var option = element5( '#header #option' );
top.EquipAfter( option );

Clone( deep )

Clone this element. This method has one param.

  • deep:boolean - The flag specify deep clone. If true, clone element's attributes and desecondaries

How to use:

var option = element5( '#header #option' );
option.Clone( false );

Animation Modifier

EffectedBy( timeline, duration, delay, timing )

Animation( timeline, duration, delay, timing )

Run( timeline, duration, delay, timing )

Play( timeline, duration, delay, timing )

Effects the animation to element. This method has four params.

  • timeline:timelineObj - The timeline object.
  • duration:number - In second, the duration of animation. CSS3 params
  • delay:number - In second, the delay time before start animation. CSS3 params
  • timing:function - The time function of animation. CSS3 params
