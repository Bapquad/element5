style5 Object

style5.AddLine( selector )

style5.AddCss( selector )

Find or create the elements with the description in selector.

  • selector - Selector string of searching.

How to use:

style5.AddLine( 'body' ).css( { background: '#000', color: '#f00' } );

How to use alt-method:

style5.AddCss( 'body' ).css( { background: '#000', color: '#f00' } );
