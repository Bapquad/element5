element5 Methods

element5.Create( target )

Create the element with the description in target.

  • target - Description for creating.

How to use:

var el0 = element5.Create( 'div#wrapper' );
var el1 = element5.Create( 'div.wrapper' );
var el2 = element5.Create( 'div[id=wrap][class=layout]' );

element5.GetBody()

Receives the body element. Notes with the DOMContentLoaded event.

How to use:

window.addEventlistener( 'DOMContentLoaded', function( e ) 
{
    var bodyElement = element5.GetBody();
}, false );

element5.GetWindow()

Receives the window.

How to use:

element5.GetWindow().addEventlistener( 'DOMContentLoaded', function( e ) 
{
    var bodyElement = element5.GetBody();
}, false );

element5.MountDevice( description )

Mounting the specify media device which your apps will run.

  • description - Description about device.

How to use:

var mobile400 = 
{
	deviceOption: '', 
	deviceType: 'screen', 
	features: [ 'max-width:399px' ], 
	producted: false,
};
element5.MountDevice( mobile400 );

element5.Extension( extObj )

Helps to creating the extend functions for DOM element. Notes it help to extends, but not help to override.

  • extObj - Object contains the methods used to manipulating on DOM elements.

How to use:

element5.Extension( {
    sayHello: function() 
    {
        console.log( 'Hello! I\'m an elements.' );
    },
});
var dom = element5.Create( 'h1' );
dom.sayHello();
